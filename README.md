## Where to Find My Answers and Explanations

I have completed all the exercises directly within the notebook. You can find my detailed answers, explanations, and the corrected code by opening the [`debugging.ipynb`](debugging.ipynb) file.

For each exercise, please look for the Markdown cells titled:
**`# Exercise <N> - Solution and Explanation`**

These sections contain:

- **Explanations:** My analysis of why the original code failed and the reasoning behind the fixes.
- **Fixed Code:** The corrected Python code blocks implementing the solutions (usually marked with `======================== FIX ========================`).

- **Exercise 1:** Fixing an issue with Python `set` non-deterministic ordering.
- **Exercise 2:** Resolving a bug related to NumPy sequential array assignments and swapping coordinates.
